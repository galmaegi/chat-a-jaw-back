# chat-a-jaw-back

### intellij
`File -> New -> Project from Version Control`   
git url 입력, directory 원하는 곳으로 설정   
`clone`!   

다운로드 된다음 알림창에서   
`checkout`!   
   
켜지면서 오른쪽 아래에 뜨는 알림창에서    
`import gradle project`!   

import 완료되면, 오른쪽 상단에   
ChatAJawApplication `Run` ▶️ !    

브라우저에서 `localhost:8080` 접속   
SampleController에 설정된 `sample` 문자열 확인!   

끝!
